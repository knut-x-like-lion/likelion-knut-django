멋쟁이 사자처럼이 무엇인가요?

"멋쟁이 사자처럼"은 비전공자를 대상으로 프로그래밍을 가르치는
대학 연합 교육 단체입니다.

멋쟁이 사자처럼의 목표

컴퓨터를 모르는 학생들이 프로그래밍 교육을 통해 원하는 IT 서비스를 직접 런칭합니다.
다양한 오프라인 활동과 병행하여 뜻을 같이하는 동료를 찾고 자신이 만들고자 하는
서비스를 직접 프로그래밍 하도록 돕습니다.

  • 최신 기술 스택 활용
  • 해커톤을 통한 아이디어 개발/서비스 구현
  • 자체제작 플랫폼을 통한 개별 학생 관리
  • 학교/팀별 온/오프라인 교육 및 스터디, 세미나

우리가 배우는 것들

HTML5/CSS3

HTML은 웹을 이루는 가장 기초적인 구성 요소입니다.
웹페이지의 내용을 서술하고 정의하는 데 사용합니다.
CSS는 웹 페이지의 모양과 표현을 서술합니다. - MDN

Python

Python은 가장 배우기 쉬운 프로그래밍 언어 중 하나며,
우리는 이 언어를 통해 사용자의 요청에
동적으로 응답할 것입니다.

Git

Git은 가장 많이 쓰이고 있는
버전 관리 도구(Version Control System)입니다.
Git을 통한 파일 관리와 백업의 혁신에 대해 배웁니다.

Django

Django는 웹 개발 업무를 빠르고 쉽게 진행하도록
설계되어웹 프레임워크를 통해 웹 사이트를
체계적이고 깔끔하게 관리할 수 있습니다.

JavaScript (예정)

JavaScript는 웹브라우저에
내장된 프로그래밍 언어입니다.
